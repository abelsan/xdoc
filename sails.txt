Sails.JS

cli commands
------------


1) Download from github (I downloaded zip files)
https://github.com/LearningBoard

2) Unzip files. You should have two folders: node-backend-master, and prototype-master

3) Install grunt globally
npm install -g grunt-cli

4) Install grunt locally
npm install grunt --save-dev

5) Open a command window, move to the folder and run npm install (it will take a long time)
npm install

6) to run the application
sails lift
